### 作業二、自動化操作

- 目標網頁：愛食記 <https://ifoodie.tw/>
- 操作流程：
  1. 自動選取城市為台北市
  2. 操控滑鼠拖曳網頁 Google Map 至學校附近區域
  3. 刷新地圖搜尋指定區域店家
  4. 爬取推薦內容
  5. 印出結果並自動換頁
- 爬取方法：
  - 使用 Selenium, Requests, BeautifulSoup, Pyautogui 套件
  - 運用 Chrome Webdriver
- 作業成果：<https://drive.google.com/file/d/1vuncAwKvecaTqMLtuzEfac2uvIxHCfJr/view>
